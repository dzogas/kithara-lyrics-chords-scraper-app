# Kithara Lyrics & Chords Scraper (Android)

Ένα Android app που φέρνει στο κινητό σου τη λειτουργικότητα του Chrome extension για το **kithara.to**.  
Με αυτό μπορείς να κάνεις scraping στίχους και ακόρντα από τραγούδια, να τα μετατρέπεις σε **ChordPro format**, και να τα αποθηκεύεις ή μοιράζεσαι άμεσα.

---

## ✨ Χαρακτηριστικά
- 📥 Scraping στίχων και ακόρντων από το kithara.to
- 🎼 Text export με σωστή inline τοποθέτηση ακόρντων
- 📂 Αποθήκευση αρχείων με `.txt` extension
- 📋 Clipboard automation για γρήγορη αντιγραφή/επικόλληση
- 🌍 Πλήρης ελληνική τοπικοποίηση στο UI
- 📱 Εύκολο μοίρασμα μέσω Android intents (π.χ. email, messenger, cloud)

# 📱 Οδηγίες Εγκατάστασης Νο.1

## Πώς να εγκαταστήσετε το APK από το GitHub Release

### 1. Κατεβάστε το APK
- Μεταβείτε στη σελίδα του GitHub repository.
- Πηγαίνετε στην ενότητα **Releases**.
- Βρείτε την τελευταία έκδοση (π.χ. `v0.1.1`).
- Κατεβάστε το αρχείο με όνομα `app-release.apk`.

### 2. Ενεργοποιήστε την εγκατάσταση από άγνωστες πηγές
- Στη συσκευή Android, ανοίξτε **Ρυθμίσεις** → **Ασφάλεια**.
- Ενεργοποιήστε την επιλογή **Εγκατάσταση εφαρμογών από άγνωστες πηγές** ή **Install unknown apps**.
- Στις νεότερες εκδόσεις Android, δώστε άδεια συγκεκριμένα στην εφαρμογή (π.χ. Chrome ή File Manager) που θα ανοίξει το APK.

### 3. Εγκαταστήστε το APK
- Ανοίξτε τον **File Manager** ή τον **Browser** όπου κατεβάσατε το APK.
- Πατήστε πάνω στο αρχείο `app-release.apk`.
- Επιλέξτε **Εγκατάσταση (Install)**.
- Περιμένετε μέχρι να ολοκληρωθεί η διαδικασία.

### 4. Εκκινήστε την εφαρμογή
- Μετά την εγκατάσταση, θα εμφανιστεί το εικονίδιο της εφαρμογής στο μενού.
- Πατήστε το εικονίδιο για να ξεκινήσετε να χρησιμοποιείτε την εφαρμογή.

---

## ⚠️ Σημειώσεις
- Αν δείτε μήνυμα όπως **“Blocked by Play Protect”**, επιλέξτε **Install anyway**.
- Κατεβάζετε πάντα το APK από το **επίσημο GitHub release** για ασφάλεια.
- Αν η εγκατάσταση αποτύχει, βεβαιωθείτε ότι η έκδοση Android της συσκευής σας υποστηρίζεται από την εφαρμογή.

# 🚀 Οδηγίες Εγκατάστασης Νο.2

## Πώς να εγκαταστήσετε την εφαρμογή με το Android Studio
1. Κατέβασε το project ή κάνε clone:
   ```bash
   git clone https://github.com/USERNAME/kithara-lyrics-chords-scraper.git
2. Άνοιξε το project στο Android Studio.
3. Κάνε build και τρέξε το app σε emulator ή συσκευή (θα πρέπει να την έχετε συνδέσει στον υπολογιστή σας με USB).

## 🛠 Τεχνολογίες
- Kotlin / Android SDK
- Gradle
- ChordPro format parsing
- Localization (Greek)

---

## 📌 Σημειώσεις
- Το app είναι **single-purpose**: μόνο για scraping και export, χωρίς περιττές λειτουργίες.
